# Gestión de Clientes y Pagos de Terrenos

## Descripción
Este proyecto consiste en una aplicación en desarrollo para la recopilación y gestión de datos de clientes a los que se les han vendido terrenos bajo un esquema de pagos mensuales. El sistema busca llevar un control ordenado de la información de los clientes, los abonos realizados y los saldos pendientes.

Actualmente, el proyecto se encuentra en una fase inicial, enfocada en el aprendizaje de conceptos básicos de programación y el manejo de datos.

## Motivación
La motivación de este proyecto surge de una necesidad real de organizar y administrar de forma eficiente los pagos mensuales de clientes que han adquirido terrenos. Además, este proyecto tiene como objetivo reforzar mis conocimientos en programación y bases de datos, aplicándolos a un caso práctico del mundo real.

Este proyecto también sirve como una introducción personal al desarrollo de sistemas de gestión, permitiéndome aprender de forma progresiva y estructurada.

## Tech Stack
Este proyecto se encuentra en una etapa de aprendizaje, por lo que el uso de tecnologías avanzadas se incorporará de manera gradual conforme se adquieran más conocimientos en programación y bases de datos.
